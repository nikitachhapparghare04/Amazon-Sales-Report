**Project Title**
Amazon-Sales-Report

**Short Description / Purpose** 
This report analyzes Amazon sales data to understand overall performance across products, categories, locations, and order statuses. It highlights key trends such as top-selling items, best-performing states and cities, customer retention levels, and delivery efficiency. The purpose of this report is to identify growth opportunities, improve logistics, optimize inventory, and support data-driven business decisions for increasing sales and customer satisfaction.

**Tech Stack**
The Dashboard was built using the following tools and technologies:
1. Power BI Desktop - Data cleaning, transformation, and modeling, Creating interactive dashboards and visualizations, DAX measures and calculated columns.
2. Power Query - Data extraction, loading, and transformation (ETL), Removing duplicates, filtering, merging, and shaping data.
3. DAX (Data Analysis Expressions) - Creating calculated measures, calculations (Total Sales, Quantity, Retention, Top States, etc.
4. Microsoft Excel / CSV Files - Source files for sales, orders, customers, and product data.
5. Visualization Tools - Bar charts, Line charts, Donut charts, Maps, Cards, Slicers, Cards, Custom icons and formatting for UI enhancement

**Features of the Report**
1. Interactive Sales Overview - Real-time KPIs showing Total Sales, Total Quantity Sold, and Total Orders.
2. Product Performance Analysis - Category-wise and size-wise sales performance, Top-selling products by quantity and revenue.
3. Order & Courier Status Tracking - Breakdown of delivered, cancelled, returned, and in-transit orders, Courier performance insights.
4. Geographical Sales Analysis - State-wise and city-wise sales distribution using maps and bar charts, Identification of top-performing regions.
5. Customer Retention Monitoring - State-level customer retention percentage visualization.
6. Dynamic Filtering - Filters for Category, Status, Size, Quarter, and Sales Channel for drill-down analysis.
7. Visually Appealing UI - Custom icons, theme-based layout, and clean visual structure.
8. DAX Calculations - Measures for sales, quantity, retention, and performance KPIs.
   
**Highlights of the Report**
1. T-shirts, shirts, and blazers are the top revenue-generating categories.
2. Karnataka and Bengaluru lead in overall sales contribution.
3. Sizes M, L, and XL have the highest demand across orders.
4. 25% cancellation rate indicates an improvement area in logistics or customer experience.
5. States like West Bengal and Uttar Pradesh show 100% customer retention, highlighting strong loyalty.
6. Underperforming categories like Wallets, Socks, and Perfume need strategy improvements.
7. The dashboard enables data-driven decision-making with clear visuals and easy-to-use slicers.
